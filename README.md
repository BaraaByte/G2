# G2 Programming Language 🚀

**G2** is a modern programming language designed for **speed, simplicity, and developer productivity**.
It is built from scratch with its own **compiler / interpreter**, clean syntax, and a lightweight runtime.

G2 aims to combine the **simplicity of scripting languages** with the **performance of compiled languages**.

---

# ✨ Features

* 🚀 Fast and lightweight
* 🧠 Simple and readable syntax
* ⚙️ Custom compiler / interpreter
* 🔧 Easy to learn
* 📦 Built-in standard library
* 💻 Cross-platform support (Windows / Linux / macOS)

---

# 📜 Example

```g2
func main() {

    name = input("Enter your name: ")

    print("Hello " + name)

    for i in 1..5 {
        print(i)
    }

}
```

---

# 🏗 Architecture

G2 works using the following pipeline:

```
Source Code (.g2)
       ↓
Lexer
       ↓
Parser
       ↓
AST (Abstract Syntax Tree)
       ↓
Interpreter / Compiler
       ↓
Program Execution
```
